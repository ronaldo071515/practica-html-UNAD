Etiquetas HTML

# Etiqueta Descripción
1 <!DOCTYPE html> Se trata de una etiqueta que no necesita cierre y cuya función es facilitar información al servidor web que aloja página. La información facilitada por esta etiqueta se refiere al tipo de documento, además es necesaria para la comunicación entre el navegador y el servidor
2 <html></html> Se trata de una etiqueta par y es la etiqueta que encierra todo el documento HTML
3 <body></body> Es una etiqueta par y la conocemos como el cuerpo del sitio web. Todo lo que se escriba dentro de esta etiqueta será visible por los navegadores.
4 <head></head> Etiqueta para cabezote de la página
5 <nav></nav> Etiqueta para barra de navegación
6 <section></section> Etiqueta para crear secciones
7 <h1></h1> Etiquetas para crear titulos
8 <p></p> Etiquetas para crear parrafos
9 <br> Etiqueta para crear saltos de linea, no se cierra
10 <a href=""></a> Etiqueta para crear vinculos
11 <img src="" alt="" src=""> Etiqueta para contener imagenes en nuestro sitio web
12 <div></div> Etiqueta contenedor


Estilos que pueden aplicarse en CSS

# Etiqueta Descripción
1 color: blue; Define el color del texto. El color deseado puede escribirse directamente, en inglés, o puede utilizarse el sistema RGB en hexadecimal. En el ejemplo el texto es de colore azul.
2 font-style: italic Solamente tiene dos posibilidades: normal (por defecto) o italic, y que hace que el texto tenga cierta inclinación.
3 font-size: 10px También muy conocido: define el tamaño de los caracteres. Además del tamaño definido por el usuario, expresado en cualquiera de las unidades de medida conocidas, puede tener alguno de los siguientes valores absolutos: xx-large x-large large medium small x-small xx-small.
4 background: green; para definir las propiedades del fondo de un elemento. En este caso "green"
5 border: #ddd; para definir las propiedades del borde de un elemento, en este caso el borde de nuestra tabla sera:
#ddd
